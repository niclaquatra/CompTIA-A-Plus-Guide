Welcome to the CompTIA A+ Exam Study Guide

A simple study guide I built during my time preparing for the exam.

Core 1 (220-1201)

CompTIA A+ Core 1 focuses on the fundamental IT concepts, including hardware, networking, and mobile devices. This exam validates your ability to identify, use, and troubleshoot hardware, mobile devices, and networking components. You'll need to demonstrate your hands-on understanding of these critical IT infrastructure components.

Main Topics Covered:
  • Hardware - CPU, RAM, storage devices, cables and connectors, motherboards and power
  • Networking - TCP/IP, ports, protocols, wireless networking, SOHO networks
  • Mobile Devices - Laptops, tablets, smartphones, mobile device configuration
  • Operating Systems - Windows installation, command-line tools, control panel
  • Security - Physical security, authentication, wireless security, malware
  • Troubleshooting - Hardware issues, networking problems, mobile device issues

Core 2 (220-1202)

CompTIA A+ Core 2 concentrates on operating systems and security, with an emphasis on the skills needed for IT operational procedures. This exam tests your knowledge of installing and configuring operating systems, software troubleshooting, and security concepts. You'll need to understand both technical skills and operational procedures used in the IT field.

Main Topics Covered:
  • Operating Systems - Advanced Windows configuration, Linux, macOS, mobile OS
  • Security - Active Directory, physical security, social engineering, data handling
  • Software Troubleshooting - OS issues, security problems, mobile OS difficulties
  • Operational Procedures - Documentation, change management, disaster recovery, safety
